## Getting Started

### Dependencies

* MS SQL SERVER
* PHP 8.1
* SQL SRV DRIVER (Both PDO & non-PDO)

### Installing

* Pull from github
* run commands

### Commands

Install after pull

```
composer install
copy .env.example .env
php artisan storage:link
php artisan key:generate
php artisan ide-helper:generate
npm install
npm run dev
```
